# Selenium Automation Testing Framework

A scalable and maintainable UI Automation Testing Framework developed for the **Automation Exercise** e-commerce website using **Java, Selenium WebDriver, TestNG, Maven, and the Page Object Model (POM)**.

The project was initially developed as part of my automation testing practice and later **refactored** to improve its architecture, code quality, maintainability, and scalability by applying automation testing best practices.

---

# Project Overview

This framework automates **26 End-to-End test scenarios** covering the core functionality of the Automation Exercise website, including:

* User Registration
* Login & Logout
* Product Search
* Product Details Verification
* Add Products to Cart
* Shopping Cart Management
* Checkout Process
* Product Quantity Validation
* Categories & Brands
* Product Reviews
* Recommended Products
* Subscription
* Address Verification
* Invoice Download
* Scroll Up / Scroll Down Functionality
